# newsRecommender
teamwork

新闻推荐系统，该系统目标是针对不同用户浏览的新闻，建立起用户模型，并随之用户的阅读习惯不断更新该用户模型，经过用户与该系统一段时间的磨合，最终对用户进行精准的新闻推荐。
